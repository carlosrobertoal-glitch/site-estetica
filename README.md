# Arlete Centro de Estética

Este repositório contém o site do **Arlete Centro de Estética**, desenvolvido para apresentar os serviços de massagem linfática e limpeza de pele em Porto Velho.

## 📂 Estrutura do projeto
- `index.html` → Página principal do site
- `logo_ciec.png` → Logo da clínica
- `atendimento.jpg` → Imagem de atendimento
